---

## 📌 Overview

This repository contains a single-page portfolio built from scratch with clean HTML, custom CSS animations, and vanilla JavaScript interactions.

It highlights:
- Professional profile and domain focus (**Full-Stack + IoT + Embedded**)
- Verified IBM/Credly certifications
- Skills across software + hardware integration
- Real projects with live deployment links
- Internship/contact availability

---

## ✨ Features

- 🎨 Premium glassmorphism UI with animated gradient orbs
- 📱 Responsive design for desktop and mobile
- 🧭 Sticky smooth-scroll navigation
- 👀 Scroll reveal animations using `IntersectionObserver`
- 🖱️ Lightweight mouse-parallax visual effects
- 🧩 Sectioned content architecture (Hero, Skills, Projects, Contact)
- 🔗 Direct links to GitHub, LinkedIn, Credly, and live projects

---

## 🧱 Sections Included

1. **Hero** – Name, role, summary, quick links, and highlights  
2. **Defense Tech Alignment** – Domain-fit statement for defense roles  
3. **Certifications** – IBM/Credly badge cards  
4. **Skills** – Languages, frameworks, databases, IoT protocols, tools  
5. **Projects** – Featured and deployed work with tech tags  
6. **Contact** – Email, phone, LinkedIn, GitHub, location  

---

## 🚀 Featured Projects Showcased

- **ChainVista** – Supply chain platform with blockchain integrity + IoT simulation  
- **MusicGlow** – Spotify-style streaming app with OAuth2 & RBAC  
- **Course Suggester** – Recommendation system for academic pathways

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Styling:** Custom CSS, gradients, blur effects, responsive media queries
- **Animations:** CSS keyframes + IntersectionObserver + JS parallax
- **Deployment:** GitHub Pages
